jniscrteng
==========
javaのScriptEngineを利用して、動的プログラム(jruby, jython3, graaljs)内の関数及びメゾッドの呼び出す  

|directory   |内容　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　|
|------------|---------------------------------------------------------------------------------|
|[lib](lib)  |C言語からScriptEngineを利用して、動的プログラム内の関数及びメゾッドを呼び出すライブラリ|
|[test](test)|上記ライブラリのtestプログラム                                                     |
