rubytool
========
1. 目的

   c言語からrubyのライブラリを呼び出すことができる

1. 前提

   サポートOS: linux
   サポート言語: ruby

1. 機能範囲

   ![ruby tool](images/ucRubyTool.jpg)

1. 機能範囲外

   vine linuxに付属しているサポート言語以外は、できない
