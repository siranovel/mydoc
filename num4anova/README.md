1. 機能詳細
    * 多重比較
      * パラメトリック検定
        - turkeyの方法による多重比較
        - ボンフェロー二の不等式による多重比較(T検定)
        - Dunnet検定
          - 両側検定
          - 左側検定
          - 右側検定
      * ノンパラメトリック検定
        - ボンフェロー二の不等式による多重比較(T検定)
    * 分散分析
      * 一元配置の分散分析
        - 箱ひげ図

        - 一元配置分散分析
        - バートレット検定

      * 二元配置の分散分析
        - 二次元分散分析(繰り返し数が等しい時)
        - 二次元分散分析(繰り返しなし)
      * 共分散分析
        - 回帰直線の平行性の検定
        - 回帰直線の有意性検定
        - 水準間の差の検定
